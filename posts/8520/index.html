#git#
subversion——集中型版本管理系统

git——分散型版本管理系统

Git 基本命令

1、git init —— 初始化仓库
2、git status —— 查看仓库状态
3、git add —— 向暂存区中添加文件
4、git commit —— 保存仓库的历史记录
5、git log——查看提交日志
6、git diff——查看更改前后的差别
7、git checkout -b [name] —— 以当前 master 分支创建新的分支

Pull Request

Pull Request 是自己修改源代码后，请求对方仓库采纳该修改时采取的一种行为。
