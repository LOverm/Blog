路由中有三个基本的概念 route, routes, router。

1、 route，它是一条路由，由这个英文单词也可以看出来，它是单数， Home 按钮 => home 内容， 这是一条 route, about 按钮 => about 内容， 这是另一条路由。

2、 routes 是一组路由，把上面的每一条路由组合起来，形成一个数组。[{home 按钮 =>home 内容 }， { about 按钮 => about 内容}]
